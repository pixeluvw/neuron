# Changelog

All notable changes to the Neuron CLI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-05

### Added
- Initial release of Neuron CLI
- `neuron create` command to generate new projects
- `neuron generate screen` command to create screens with controllers
- `neuron generate controller` command
- `neuron generate model` command
- Support for empty projects and organization names
